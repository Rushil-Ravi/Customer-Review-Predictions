# Comprehensive Regression Modeling for Customer Review Predictions

## Overview
This project analyzes TripAdvisor hotel review data to understand factors influencing
customer satisfaction and hotel ratings using regression-based modeling techniques.

## Dataset
- Source: UCI Machine Learning Repository (Las Vegas TripAdvisor Dataset)
- Records: 504
- Features: 18 (categorical + numerical)

## Methods
- Exploratory Data Analysis (EDA)
- Linear & Multiple Linear Regression
- Polynomial Regression
- Logistic Regression (classification)
- Regularization: Ridge, Lasso, Elastic Net
- Dimensionality Reduction: PCA

## Key Results
- Polynomial regression captured non-linear relationships effectively
- Amenities and traveler type strongly influenced ratings
- Logistic regression enabled high vs low rating classification

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
